- Exact duplicate receipt images were detected across sources using SHA-256 hashing and deduplicated prior to release.

### Data completeness

This dataset reflects real-world receipt quality. Some fields may be missing due to
blur, cropping, low resolution, or incomplete prints.

**Merchant fields**
- Website missing or empty in **84.5%** of samples
- Name missing or empty in **20.0%**
- Address missing or empty in **28.5%**
- Phone missing or empty in **51.3%**
- Tax ID missing or empty in **37.4%**

**Transaction fields**
- Payment method missing or empty in **23.7%**
- Date missing or unknown in **26.9%**
- Time missing or unknown in **31.5%**
